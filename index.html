Goodbye, schemas and SQL. Hello, JSON and MapReduce. CouchDB is a database built for the web.

You’re designing for the modern world, where needs change fast. Your app has be up quickly, and updatable at a moment's notice. Oh, and it has to scale to a multi-terabyte cluster.

For changing demands, you need agile tools. But that relational database you’re using — it chops up your business objects, it bites you when you shake things up, and SQL doesn’t complement your scripting language.

You want a database that understands objects. You want to talk to it in your native language. It should accept the odd new attribute without balking. And it should offer all the scalability and reliability that kept you hanging onto that RDBMS.

Enter CouchDB.

Flexible

Start saving your data today, then adapt your queries as your application evolves, without expensive schema migrations. Query your data using familiar JavaScript (or your language of choice). CouchDB's incremental MapReduce views are optimized for real-time workloads, not overnight batch jobs.

Web scale

CouchDB scales from a smartphone to a data-center, and beyond. The Erlang and REST-based implementation presents the same, simple API whether running on a small local instance, an ad-hoc network of collaborators, or a multi-terabyte eventually-consistent cluster.

Reliable

CouchDB uses append-only storage for extreme durability. It never touches any bytes already written to disk, so even truncating a CouchDB database file yields a consistent snapshot of an older version of the database. Writes are contiguous, giving CouchDB predictable performance even under heavily concurrent write load. The MVCC data model is ideal for concurrent operations that require consistency.

Ground computing

CouchDB solves the data-island problem, so you can write offline capable applications. Peer-based replication allows for flexible deployments, data-sharing between organizations, and no-hassle backups. CouchDB on the client offers better latency and reliability than traditional three-tier architectures. Ground computing is the future of the web.

Trusted

CouchDB is deployed by the BBC, Meebo, Cloudant and Ubuntu One. It ships as a core feature of Ubuntu Linux. There are access libraries available in nearly every language.
